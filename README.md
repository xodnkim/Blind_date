# Blind Date (블라인드 데이트)

프리미엄 블라인드 데이트 매칭 서비스 프로젝트입니다.

## 🚀 시작하기

1. 이 저장소를 클론합니다.
2. `index.html`을 브라우저에서 엽니다.

## 🛠️ 기술 스택
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Design**: Glassmorphism, Responsive Design
- **Future Integration**: Supabase (Auth, Database)

## 📦 깃허브 연결 방법
이미 로컬에서 `git init`이 완료되었습니다. 다음 명령어를 통해 자신의 깃허브에 올릴 수 있습니다:

```bash
git add .
git commit -m "Initial commit: Premium landing page"
git remote add origin https://github.com/사용자이름/Blind_date.git
git branch -M main
git push -u origin main
```
